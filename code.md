##기본작성법
파일확장자는 .msl입니다.
명령어 작성은 소문자를 기준으로하나 메세지 입력 부분에서는 대문자를 허용합니다.
파일 최상단에는 다음과 같은 코드가 있어야 합니다.
```java
start magicspell-metatron
```
공백은 금지되어 있으며 한줄을 비워도 안됩니다.
▼금지된 행동▼
```java
 w r o n g c o d e
 
```
명령어 작성시 맨앞에 공백이 있어도 안됩니다.
▼금지된 행동▼
```java
 start magicspell-metatron
```
▼금지된 행동▼
```java
start magicspell-metatron//스팰시작
 
```
